# AddNewTrip

'새로운 여행 추가하기'기능에서 '날짜 추가하기' 뷰에 해당하는 자료입니다.

캘린더를 라이브러리를 사용하지 않고 TableView와 CollectionView를 활용해 구현하는 법을 연습했던 자료입니다.

테이블뷰 셀안에 컬렉션뷰가 포함되는 구조입니다.
